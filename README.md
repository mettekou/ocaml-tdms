# tdms

TDMS \195\162.0 file support for OCaml

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Tdms.greet "World"
```
